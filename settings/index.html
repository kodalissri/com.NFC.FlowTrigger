  1. Get an NFC app: Install an NFC app like "NFC Tools" (iOS/Android) or "Trigger" (Android)
  2. Write to NFC tag: In the app, create a new task that sends a web request (HTTP/HTTPS)
  3. Configure the URL: Use the webhook URL above and add ?tag_id=YOUR_TAG_ID at the end
  4. Example: https://your-homey/api/app/com.yourdomain.nfc-flow-trigger/trigger?tag_id=bedroom
  5. Write to tag: Hold your phone to the NFC tag to write the URL
  6. Add tag below: Add the same tag ID in the form below to give it a friendly name
  7. Create flows: Use the "NFC tag scanned" flow cards in Homey to trigger actions
