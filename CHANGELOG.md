# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - TBD

Initial release of NFC Flow Trigger for Homey Pro.

### Features
- NFC tag triggering via webhook endpoint
- Tag configuration and management through settings page
- Two flow trigger cards:
  - "NFC tag scanned" - Triggers for any tag with optional filter
  - "Specific NFC tag scanned" - Triggers for specific configured tags
- Mobile phone integration with NFC apps (iOS/Android)
- Friendly tag naming system
- Public API endpoint for easy integration
- Flow tokens for tag ID and tag name

### Setup
- Works with NFC Tools app on iOS and Android
- Webhook-based architecture for maximum compatibility
- Simple web-based settings interface
